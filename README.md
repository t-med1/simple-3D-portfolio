### Run Locally

To run **3D Portfolio** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/t-med1/simple-3D-portfolio.git
```

Windows:

```bash
git clone https://github.com/t-med1/simple-3D-portfolio.git
```
### Run Project

- First : `npm install`
- Second : `npm run dev`
